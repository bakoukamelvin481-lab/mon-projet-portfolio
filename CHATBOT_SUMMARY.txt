🚀 MISE EN PLACE (QUICKSTART)
=============================================================================

ÉTAPE 1: Obtenir une clé API OpenAI
────────────────────────────────────
  1. Va sur: https://platform.openai.com/account/api-keys
  2. Crée compte (gratuit avec crédit d'essai $5)
  3. Clique "Create new secret key"
  4. Copie la clé (ex: sk-proj-abc123...)

ÉTAPE 2: Ajouter la clé au chatbot
──────────────────────────────────
  Ouvre: chatbot/config.js
  
  Remplace:
  apiKey: 'sk-YOUR_API_KEY_HERE',
  
  Par:
  apiKey: 'sk-proj-abc123...',  ← ta vraie clé
  
ÉTAPE 3: Tester
───────────────
  1. Rafraîchis ton site (Ctrl+F5)
  2. Clique sur le 💬 en bas à droite
  3. Pose une question sur BFA
  4. L'IA répond en français!

💰 COÛTS ESTIMÉS
=============================================================================

GPT-3.5-Turbo (recommandé):
  - Coût: ~$0.50 par 1M tokens
  - 10 conversations/jour = ~$1-3/mois
  - 100 conversations/jour = ~$10-30/mois
  
Track: https://platform.openai.com/account/billing/overview

CONSEIL: Utilise le fallback pour les questions fréquentes!

🔐 SÉCURITÉ (PRODUCTION)
=============================================================================

⚠️ NE PAS:
  ❌ Laisser la clé API en public
  ❌ Committer config.js dans Git
  ❌ Afficher la clé au client

✅ À FAIRE:
  1. Utiliser un backend (Node.js, Python, etc.)
  2. Mettre la clé en variable d'environnement
  3. Rate-limiter les appels
  4. Monitorer les coûts

Guide complet: Voir CHATBOT_GUIDE.md
